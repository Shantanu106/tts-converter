## 🚀 HOW TO USE

### Option 1: Fastest Start (Recommended)

**Windows:**
```bash
cd "c:\Users\shant\Desktop\day 2"
quickstart.bat
```

**Linux/Mac:**
```bash
cd "c:\Users\shant\Desktop\day 2"
bash quickstart.sh
```

Then open: `http://localhost:5173`

### Option 2: Manual Setup

**Terminal 1 - Backend:**
```bash
cd "c:\Users\shant\Desktop\day 2\backend"
python -m venv venv
venv\Scripts\activate  # or: source venv/bin/activate (Mac/Linux)
pip install -r requirements.txt
python main.py
```

**Terminal 2 - Frontend:**
```bash
cd "c:\Users\shant\Desktop\day 2\frontend"
npm install
npm run dev
```

**Browser:** `http://localhost:5173`

### Option 3: Docker

```bash
cd "c:\Users\shant\Desktop\day 2"
docker-compose up
```

Then:
- Backend: `http://localhost:8000`
- Frontend Dev: `http://localhost:5173`
- Frontend Prod: `http://localhost` (with nginx)
